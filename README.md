XSS_Payload_GA_Generator
========================
XSS payload generator by using GA(Genetic Algorithm)

## Requirements
- Python 3.11.2


## Usage
run generator
```
python3 generator.py
```

run tensorboard
```
tensorboard --logdir='./log' --bind_all
```

## Example
```
python3 generator.py
```
